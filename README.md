# Publication2  
## Machine Learning & Statistical Analysis of Entrepreneurship and Social Media Impact

---

## Overview
This repository contains **datasets, machine learning notebooks, clustering analyses, and supporting research documents** focused on **entrepreneurship prediction and the role of social media in economic growth**, with special emphasis on **female and youth entrepreneurship in Pakistan**.

The work combines **statistical analysis, PCA, clustering, and supervised machine learning**, making it suitable for **research publications, MSc/PhD portfolios, and policy-driven data science studies**.

## Datasets Description

### Entrepreneurship Prediction Dataset
- Format: CSV
- Purpose:
  - Feature engineering
  - Classification of entrepreneurial intent
  - Model training and evaluation

### Female Entrepreneurship Regional Data (Pakistan)
- Provinces:
  - Punjab
  - Sindh
  - KPK
  - Islamabad
  - Balochistan & AJK
- Format: Excel (`.xlsx`)
- Use:
  - Regional statistical comparison
  - Gender-based economic participation analysis

### GEM 2021 NES Regional Dataset
- Source: Global Entrepreneurship Monitor (GEM)
- Level: Regional indicators
- Use:
  - Cross-regional entrepreneurship analysis
  - Policy-oriented research

### Entrepreneurial Competency Dataset
- Cleaned and labeled survey data
- Binary subsets (`Yes / No`)
- Use:
  - Classification tasks
  - Competency and behavioral analysis

## Machine Learning & Analysis Notebooks

### `entrepreneurship_ML.ipynb`
- Complete supervised ML pipeline:
  - Data preprocessing
  - Feature scaling and encoding
  - Model training
  - Performance evaluation and visualization

### `ClusteringPaper2.ipynb`
- Unsupervised learning and exploratory analysis:
  - PCA (2D, 3D, and 4D)
  - K-Means clustering
  - Elbow and Silhouette analysis
  - Hierarchical clustering (Ward linkage)
  - Interactive 3D visualizations
